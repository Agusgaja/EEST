# MVP panel web de tickets de mantenimiento

Interfaz React + TailwindCSS para demostrar un tablero operativo de tickets recibidos desde WhatsApp y gestionados por mantenimiento.

## Funcionalidades

- Dashboard tipo Kanban por estado: Abierto, En proceso y Resuelto.
- Tarjetas con ID, usuario, sector, categoria, subcategoria, etiqueta del dispositivo, descripcion breve, fecha y estado.
- Panel lateral con detalle completo del ticket.
- Cambio de estado con registro automatico en historial.
- Alta de observaciones internas.
- Datos simulados para validar el concepto sin backend.

## Ejecutar

```bash
npm install
npm run dev
```

Luego abrir la URL local que indique Vite.
